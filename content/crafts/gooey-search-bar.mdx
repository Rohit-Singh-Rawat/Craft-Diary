# Gooey Search Bar

A visually striking search bar component with smooth gooey morphing animations that create a fluid, playful user experience.

## Features

- 🎨 Smooth gooey morphing animations
- 🔍 Expandable search interface
- 🎭 Modern and playful design
- 📱 Responsive layout
- ⚡ Performance optimized animations

## Installation

This component uses Framer Motion for animations:

```bash
npm install motion
```

## Usage

```tsx
import { GooeySearchBar } from '@/craft/components/gooey-search-bar';

<GooeySearchBar placeholder="Search..." />
```

## Props

- `placeholder?: string` - Placeholder text for the search input
- `onSearch?: (query: string) => void` - Callback when search is submitted
- `className?: string` - Additional CSS classes

## Styling

The component uses CSS filter effects to create the gooey morphing appearance. You can customize the colors and animation timing through CSS variables or by modifying the component styles.
